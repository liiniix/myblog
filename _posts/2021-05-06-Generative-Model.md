## Generative Model
Given training data, generate new samples from same distribution.

$$\text{Training data } \sim P_{data}(x)$$
$$\text{Generated samples } \sim P_{model}(x)$$

$$\text{Want to learn $P_{model}(x)$ similar to $P_{data}(x)$}$$
